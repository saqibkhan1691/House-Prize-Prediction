# House Price Prediction

A practical Linear Regression project demonstrating how **data preprocessing techniques** like **feature scaling** and **PCA** impact model performance, using both **from-scratch** and **scikit-learn** implementations.

## Project Highlights
- Linear Regression using Normal Equation (OLS)
- Gradient Descent without scaling vs with scaling
- Principal Component Analysis (PCA) for dimensionality reduction
- Performance comparison using RMSE and R²
- Future house price prediction demo

## Key Takeaways
- Feature scaling significantly improves gradient descent convergence.
- PCA reduces redundancy and improves numerical stability in higher dimensions.
- With the same dataset and same model, preprocessing alone can change results.

## Tech Stack
Python · NumPy · Pandas · Matplotlib · scikit-learn
